Given an integer, write a function to determine if it is a power of three.

**Example 1:**

```
Input: 27
Output: true
```

**Example 2:**

```
Input: 0
Output: false
```

**Example 3:**

```
Input: 9
Output: true
```

**Example 4:**

```
Input: 45
Output: false
```

**Follow up:**
Could you do it without using any loop / recursion?

 **难度系数**    
Easy

解法：通过换底公式得出结果
```c++
class Solution {
public:
	bool isPowerOfThree(int n) {
		if (n <= 0)
			return false;
		double p = log10(n) / log10(3);//log3(x)=log10(x)/log10(3)
		return (int)p == p;
	}
};
```
