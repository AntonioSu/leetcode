
### 题目描述
Given a binary tree, return the inorder traversal of its nodes' values.

Example:
```
Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,3,2]
```
思路：二叉树非递归中序遍历，一直压栈，直到遇到空节点
```c++
class Solution {
public:
	vector<int> inorderTraversal(TreeNode* root) {
    stack<TreeNode*> sta;
    vector<int> vec;
    TreeNode* head=root;
    if(!head)
        return vec;
    //sta.push(head);
    while(!sta.empty()||head){
        if (head){
            sta.push(head);
            head=head->left;
        }
        else{
            head=sta.top();
            vec.push_back(head->val);
            sta.pop();
            head=head->right;
        }
    }
    return vec;
    }
};
```
